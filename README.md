# Анимированный таймер обратного отсчета

Современный минималистичный таймер с круговым прогресс-баром, анимацией и звуковым оповещением.

##  Внешний вид

- Минималистичный дизайн в стиле современного UI
- Фиолетовая цветовая акцентуация (#7c3aed)
- Адаптивная верстка для всех устройств
- Плавные анимации и переходы

##  Функциональность

### 1. Управление временем
- **Ввод минут**: числовое поле с ограничением от 0 до 99 минут
- **Кнопка "Старт"**: запуск таймера (становится неактивной во время работы)
- **Кнопка "Пауза/Продолжить"**: 
  - Во время работы - приостанавливает таймер
  - На паузе - возобновляет отсчет
- **Кнопка "Сброс"**: полный сброс таймера в исходное состояние

### 2. Отображение времени
- Формат MM:SS с ведущими нулями
- Обновление каждую секунду
- Крупное, четкое отображение в центре круга

### 3. Круговой прогресс-бар
- Визуализация оставшегося времени в виде SVG-круга
- Плавная анимация заполнения
- **Динамическая смена цвета**:
  - Черный (осталось > 50% времени)
  - Фиолетовый (осталось < 25%)
- Анимация пульсации при завершении

### 4. Обратная связь
- **Статусные сообщения** под таймером: "Готов", "Выполняется", "На паузе", "Завершен"
- **Звуковое оповещение** при завершении (тритональная мелодия)
- Визуальное выделение при завершении (фиолетовый цвет и пульсация)
- Блокировка ввода минут во время работы таймера

##  Технологии

- **HTML5** - семантическая разметка
- **CSS3** - современный дизайн, анимации, Flexbox
- **JavaScript (ES6+)** - чистая архитектура, классы
- **SVG** - круговой прогресс-бар
- **Web Audio API** - звуковые эффекты
